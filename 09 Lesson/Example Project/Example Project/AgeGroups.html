Age Groups

In Koding Next we strongly believe that anyone can learn how to ‘code’. We offer our programming classes for the following Age Groups:

LITTLEKODERS

BASICS LITTLEKODERS

DESCRIPTION

During this course, children will learn basic aspects of coding. Course uses interactive logical activities with support of sensory toys as well as basic computer activities. Students will learn how to make simple games and interactive digital stories using ScratchJrdeveloped by educators from Tufts University and MIT.

Age 6-7
ROBOTS LITTLEKODERS

DESCRIPTION

This entry-level course teaches children how to use simple logarithms in real life situations. Students develop logical thinking skills by teaching their robots to perform given tasks. KodingNext is the first school in Indonesia and one of the first in Asia to use revolutionary ‘coding robots’: ‘Cubetto’ and “Dash’n’Dot’.

Age 6-7

JUNIORKODERS

BASICS JUNIORKODERS

DESCRIPTION

In this entry-level course students will learn how to code simple games and interactive digital stories using Scratch. Students will develop their projects with their peers, which will teach them working in teams. Enrolling in this course will give a basic foundation needed to continue learning how to code. In the future they will be able to choose other, more advanced classes.

Age 8-15
APP&GAME JUNIORKODERS

DESCRIPTION

Mobile apps are definitely one of the most important inventions of the recent years. We all use them but not many of us know how to actually make them. The game industry is also blooming, not only when it comes to mobile games. Koding Next students will learn how to make mobile apps in relation to their hobbies and things they are interested in. This will make classes and activities interesting and students will also learn how to express themselves through coding. They will also learn objective programming and game design. They will create their own 2D and 3D games. They will build a mobile app.

Age 8-15
TECH&ART JUNIORKODERS

DESCRIPTION

Tech&Art JuniorKoders will learn basics of creating programs for robots and micro controllers/computers as well as 2D and 3D design. The set of design skills learned by taking this path is an advantage for front-end developers, website developers and crucial for game industry. The understanding of computers and technology gained in this path will give students the opportunity to explore the possibilities of Internet of Things.

Age 8-15

PROKODERS

APP&GAME PROKODERS

DESCRIPTION

This path teaches students main concepts of game creating as well as develops their algorithmic thinking and introduces objective programming.

Age 16-99
TECH&ART PROKODERS

DESCRIPTION

This path lets students realize that programming is not only about laptops and smartphones but is all around us. A micro computer can make anything smart, even your clothes!

Age 16-99
WEB&SECURITY PROKODERS

DESCRIPTION

We help student take the first steps to becoming advanced web developers, data analytics or security specialists in the future. Experience in data processing, online security and computer systems are now most valued IT skills on the market.

Age 16-99
